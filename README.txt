————— For compiling —————

The project can be compiled with QTCreator or using the terminal by typing “qmake” and “make” from this folder.

The code has been tested under Linux and Mac only


————— For executing —————

all the experiments can be executed using the esec.sh script. 

Results will be collected in two files
results_homotopy.txt for the algorithm described in Robins et al. 2011
results_immersion.txt for our new algorithm
